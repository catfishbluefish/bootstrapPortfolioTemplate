Frontend Nanodegree Resume

Udacity - Project 2

HTML, CSS, Javascript, jQuery, Google Maps API


The front page of catfistori.com was originally created as part of Udacity's Frontend Nanodegree program. This project is heavily focused on javascript and jQuery, using the append and prepend features to add content to the site through JSON rather than typing info directly into the HTML index file. This way new content can simply be added into a JSON elment and is then automatically formatted and added to the page. It also pulls location data and the Google Maps API to create an interactive map at the bottom of the page with pins for each location I've worked and studied.
